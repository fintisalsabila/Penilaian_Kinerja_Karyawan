### Projek Tugas Akhir: Sistem Penunjang Keputusan dengan Algoritma SAW

**Pengantar**
Proyek ini merupakan implementasi dari algoritma Simple Additive Weighting (SAW) dalam sebuah sistem penunjang keputusan yang dibangun menggunakan framework CodeIgniter. Sistem ini bertujuan untuk membantu dalam pemilihan karyawan terbaik berdasarkan beberapa kriteria.

**Teknologi yang Digunakan**
* **Framework:** CodeIgniter
* **Bahasa Pemrograman:** PHP
* **Database:** MySQL
* **Algoritma:** Simple Additive Weighting (SAW)
* **Lainnya:** Bootstrap untuk tampilan

**Cara Penggunaan**
1. **Instalasi:**
   * Pastikan Anda telah menginstall PHP dan database yang diperlukan.
   * Clone repository ini ke direktori htdocs atau direktori web server Anda.
   * Buat database baru dengan nama yang sesuai dan import file SQL yang disediakan.
   * Konfigurasi file koneksi database pada file `application/config/database.php`.
2. **Jalankan Aplikasi:**
   * Akses aplikasi melalui browser dengan URL http://localhost/Penilaian_Kinerja_Karyawan/login.php
3. **Penggunaan:**
   * pengguna dapat menambahkan alternatif, menentukan kriteria, dan melihat hasil perhitungan SAW.

**Struktur Folder**
* **application:** Berisi seluruh kode aplikasi CodeIgniter.
    * **controllers:** Mengandung controller untuk mengelola permintaan pengguna.
    * **models:** Mengandung model untuk berinteraksi dengan database.
    * **views:** Mengandung tampilan halaman web.
* **assets:** Berisi file statis seperti CSS, JavaScript, dan gambar.
* **database:** Berisi file SQL untuk membuat struktur database.
